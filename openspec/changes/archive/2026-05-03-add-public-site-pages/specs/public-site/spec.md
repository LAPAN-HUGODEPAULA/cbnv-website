## ADDED Requirements

### Requirement: Home Page Content
The Home page SHALL display high-impact information including the event date, location, primary call to action (CTA), and highlights of the congress.

#### Scenario: Visitor lands on Home
- **WHEN** the visitor accesses the root URL
- **THEN** they see the hero section with key event details and a "Register Now" button

### Requirement: Registration Informational Page
The Registration page SHALL provide information about prices, dates, and instructions, directing the user to an external platform or showing a "Coming Soon" message.

#### Scenario: Visitor views registration info
- **WHEN** the visitor navigates to the Registration page
- **THEN** they see the pricing table and a link to the external registration system

### Requirement: Submissions Informational Page
The Submissions page SHALL display the rules, important dates, and submission categories, without providing the submission form in this phase.

#### Scenario: Visitor views submission rules
- **WHEN** the visitor navigates to the Submissions page
- **THEN** they see the list of requirements and a "Submissions Open on [Date]" message
